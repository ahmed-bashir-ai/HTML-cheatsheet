HTML Table Reference

Table(<table>)

The table tag is used to wrap around all HTML table elements.

Table Head(<thead>)

The set of rows defining the column headers in a table.

Table body(<tbody>)

Set of rows containing actual table data.

Table Row(<tr>)

The table row container.

Table Data(<td>)

The table row container.

Table foot(<tfoot>)

Table Attributes

Column Span(colspan)

Defines how many columns a td element should span

Row Span(rowspan)

Defines how many rows a td element should span
